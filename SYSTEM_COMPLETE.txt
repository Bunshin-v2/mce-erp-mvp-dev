================================================================================
THE 7 IMMUTABLE RULES (ENFORCED BY DESIGN)
================================================================================

1. NO HARDCODED COLORS
   Use tokens: --color-critical, --color-success, --color-warning, --color-info

2. NO ARBITRARY FONT SIZES
   Use scale: text-xs (11px), text-sm (12px), text-base (13px)... text-4xl (32px)

3. NO ARBITRARY SPACING
   Use 4px scale: space-1 (4px), space-2 (8px), space-4 (16px), space-6 (24px)

4. NO FONT WEIGHTS BEYOND 700
   Only: 400 (normal), 500 (medium), 600 (semibold), 700 (bold)

5. NO TRANSITIONS BEYOND 400MS
   Only: 150ms (fast), 200ms (standard), 300ms (moderate), 400ms (smooth)

6. NO ALL CAPS EXCEPT STATUS
   Sentence case: "Total revenue", not "TOTAL REVENUE"
   Exception: CRITICAL, ACTIVE, AUDIT (status only)

7. NO RAW HTML COMPONENTS
   Use primitives: Box, Text, Button, Card, Stack, Flex, Grid
