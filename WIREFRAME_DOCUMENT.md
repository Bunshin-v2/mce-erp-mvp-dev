# MCE Command Center - Wireframe & UX Architecture
**Version:** 2.0 (Morgan Dark Glass 2026)
**Date:** January 24, 2026

## 1. Global Layout Structure
The application uses a persistent **Shell Layout** for all authenticated views.

### **1.1 Sidebar Navigation (Left Panel)**
*   **Style:** Fixed width (64px collapsed / 240px expanded), Charcoal (`#444444`) Gradient.
*   **Branding:** "Morgan" SVG Logo (Top Center).
*   **Menu Items:**
    *   **Dashboard** (Home Icon)
    *   **Projects** (Building Icon)
    *   **Tenders** (Briefcase Icon)
    *   **Financials** (Pie Chart Icon)
    *   **Documents** (Folder Icon)
    *   **Resources** (Users Icon)
    *   **Executive Cockpit** (Activity/Pulse Icon) - *Premium View*
    *   **AI Agent Console** (Bot Icon)
    *   **Settings** (Gear Icon)
*   **Footer:** User Avatar, "Logout" Button.

### **1.2 Global Header (Top Bar)**
*   **Style:** Transparent/Glass with Blur.
*   **Elements:**
    *   **Search Bar:** "Type to search..." (Global fuzzy search for projects/tenders).
    *   **Quick Actions:** "+ New Project", "+ New Tender" (Dropdown).
    *   **Notifications:** Bell Icon with Red Dot badge.
    *   **Context:** Breadcrumbs (e.g., `Home > Projects > AHS Hospital`).

## 2. Core Views & Elements

### **2.1 Main Dashboard (Home)**
**Aesthetic:** "Morgan 2026" Dark Glass (Charcoal #444444 @ 70% Opacity).
*   **Top Row: Executive KPIs (4 Cards)**
    *   **Total Revenue:** Value + Trend Line (Green).
    *   **Active Projects:** Count (White) + "Ongoing" label.
    *   **Pending Bids:** Count (Amber) + "Urgent" tag.
    *   **Risk Level:** Low/Med/High Meter (Dynamic Color).
*   **Middle Left: Project Portfolio List**
    *   **Container:** Dark Glass Panel with Neon Cyan Borders.
    *   **Header:** "Project Portfolio" title + Filter Toggles (Ongoing/Completed).
    *   **List Items:**
        *   Icon (Building).
        *   Project Name (White, Bold).
        *   Location (City Name).
        *   **Neon Laser Progress Bar** (Cyan).
        *   Contract Value (AED).
        *   Status Badge (Pill shape).
*   **Middle Right: Ongoing Tasks & Alerts**
    *   **KPI Grid:** 4 Neon Glass Cards (Total, Pending, In Progress, Completed).
    *   **Overall Progress:** Striped Animated Green Bar.
    *   **Quick Add:** Input Field + "Add" Button (Red #c21719).
    *   **Recent Tasks:** List of active alerts with Edit/Delete actions.

### **2.2 Executive Cockpit (Premium View)**
**Aesthetic:** Deep Space Dark (`#0f172a`) with Holographic Accents.
*   **Flash Report Header:** "Daily Briefing" auto-generated by AI.
*   **Financial Health Widget:**
    *   Revenue vs. Cost Area Chart (Recharts).
    *   Cash Flow Bar Chart.
*   **Risk Heatmap:**
    *   Grid view of projects colored by risk (Red/Amber/Green).
    *   Clicking a cell opens the detailed Risk Modal.
*   **AI Insights:**
    *   Chat Interface for "Ask about my portfolio".

### **2.3 Project Management Module**
*   **View 1: Project List (Grid/Table)**
    *   Sortable columns: Name, Code, Client, % Complete, Value.
    *   "Create Project" Modal Trigger.
*   **View 2: Project Detail (Single Project)**
    *   **Header:** Project Banner Image, Title, Client Logo.
    *   **Tabs:**
        *   **Overview:** Description, Scope, Key Dates.
        *   **Financials:** Budget Table, Invoiced vs. Paid.
        *   **Schedule:** Gantt Chart or Milestone Timeline.
        *   **Documents:** Folder view for this specific project.
        *   **Team:** List of assigned resources.

### **2.4 Tenders & Estimations**
*   **Bid Tracker (Kanban/List):**
    *   Columns: Opportunity, Estimating, Submitted, Negotiation, Awarded, Lost.
    *   **Element:** "Win Probability" Score (AI-calculated).
*   **Tender Detail:**
    *   Submission Deadline Countdown.
    *   Bond Value Calculator.
    *   Competitor Analysis Fields.

### **2.5 Financials Module**
*   **Invoice Ledger:**
    *   Table of all Invoices (Draft, Sent, Paid, Overdue).
    *   "Generate Invoice" Form Wizard.
*   **Cost Control:**
    *   Budget Variance Analysis (Planned vs. Actual).
*   **Timesheets:**
    *   Weekly resource cost approval grid.

### **2.6 Documents (DMS)**
*   **File Explorer Interface:**
    *   Sidebar: Folder Tree (Root > Projects > [Project Name] > Phase).
    *   Main Area: Grid of File Icons (PDF, DWG, XLSX).
    *   **Features:** Drag-and-drop upload, Version History (v1.0, v1.1).

### **2.7 AI Agent Console**
*   **Chat Interface:**
    *   "Chat with Nexus" text input.
    *   Message history (User vs. AI).
*   **Agent Status Grid:**
    *   **Agent 01 (Foreman):** Operational tasks status.
    *   **Agent 02 (Estimator):** Tender analysis status.
    *   **Agent 03 (Controller):** Financial audit status.

## 3. UI Component Library (Atomic Elements)

### **3.1 Colors (Morgan Palette)**
*   **Primary Brand:** `#c21719` (Morgan Red).
*   **Primary Background:** `#444444` (Charcoal).
*   **Secondary Background:** `#1e293b` (Deep Slate).
*   **Accents (Neon):**
    *   Cyan: `#33CCCC` (Projects/Tech).
    *   Emerald: `#10b981` (Success/Finance).
    *   Amber: `#f59e0b` (Warning/Tender).
    *   Rose: `#f43f5e` (Critical/Error).

### **3.2 Typography**
*   **Font Family:** `Inter` (Sans-serif) + `JetBrains Mono` (Data/Code).
*   **Hierarchy:**
    *   `H1`: 32px Bold (Page Titles).
    *   `H2`: 24px Bold (Section Headers).
    *   `H3`: 18px SemiBold (Card Titles).
    *   `Body`: 14px/16px Regular (Standard Text).
    *   `Caption`: 12px Medium (Labels/Metadata).

### **3.3 Interactive Elements**
*   **Buttons:**
    *   **Primary:** Solid Red (`#c21719`) with White Text.
    *   **Secondary:** Glass White (`bg-white/10`) with Hover Glow.
    *   **Ghost:** Text-only with Hover Underline.
*   **Inputs:**
    *   Dark background (`bg-slate-900`), Light Gray Border, White Text.
    *   Focus State: Red Glow (`ring-[#c21719]`).
*   **Cards:**
    *   Base: `bg-[#444444]/70` + `backdrop-blur-xl`.
    *   Border: `border-white/10` or Neon variations.
    *   Shadow: `shadow-xl`.

## 4. Data Architecture (Supabase)
*   `projects_master`: The core entity.
*   `tenders`: Pre-project opportunities.
*   `documents`: Linked to Projects or Tenders.
*   `invoices`: Linked to Projects.
*   `agents_activity`: Log of AI actions.
